## Project setup
```
npm install
```
### API Key
- Create Api Key from [OpenWeatherMap](https://openweathermap.org). 
- Enter the key on the line below in **src/components/Weather.vue**
```javascript
api_key: "YOUR_API_KEY"
```

### Compiles and hot-reloads for development

npm run serve
